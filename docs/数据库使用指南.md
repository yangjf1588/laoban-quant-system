## 表结构

### daily_kline（日K线）
```sql
id               -- 主键
ts_code         -- 股票代码，如 600000.SH
trade_date      -- 交易日期，如 2026-04-23
open            -- 开盘价
high            -- 最高价
low             -- 最低价
close           -- 收盘价
volume          -- 成交量
amount          -- 成交额（元）
pct_chg         -- 涨跌幅（%）
change          -- 涨跌额（元）
ma5             -- 5日均线
ma10            -- 10日均线
ma20            -- 20日均线
ma60            -- 60日均线
vol_ratio       -- 量比（成交额/5日均成交额）
amplitude       -- 振幅（%）
```

### market_sentiment（市场情绪）
```sql
trade_date      -- 交易日期
zt_count        -- 涨停数
dt_count        -- 跌停数
up_count        -- 上涨家数
down_count      -- 下跌家数
total_amount    -- 总成交额
market_width    -- 市场宽度（上涨占比%）
```

### stock_list（股票列表）
```sql
ts_code         -- 股票代码
name            -- 股票名称
industry        -- 所属行业
list_date       -- 上市日期
```

## 快速查询示例

### Python查询
```python
import sqlite3
import pandas as pd

DB = '~/Desktop/X/laoban-quant-system/data/database/quant.db'

# 查询单只股票的近期K线
conn = sqlite3.connect(DB)
df = pd.read_sql("""
    SELECT * FROM daily_kline 
    WHERE ts_code='600000.SH' 
    ORDER BY trade_date DESC 
    LIMIT 10
""", conn)
conn.close()

# 查询市场情绪
conn = sqlite3.connect(DB)
df = pd.read_sql("SELECT * FROM market_sentiment ORDER BY trade_date DESC LIMIT 5", conn)
conn.close()

# 今日涨停股
conn = sqlite3.connect(DB)
df = pd.read_sql("""
    SELECT d.ts_code, s.name, d.close, d.pct_chg
    FROM daily_kline d
    JOIN stock_list s ON d.ts_code = s.ts_code
    WHERE d.trade_date = (SELECT MAX(trade_date) FROM daily_kline)
      AND d.pct_chg >= 9.5
    ORDER BY d.pct_chg DESC
""", conn)
conn.close()
```

## 注意事项

1. **涨跌幅(pct_chg)**: 东方财富接口已修复，全量数据已补充
2. **成交量(volume)**: 部分股票volume为空（腾讯API限制），成交额(amount)始终有数据
3. **量比(vol_ratio)**: 基于成交额计算，用amount替代volume
4. **数据更新**: 定时采集每日16:05自动跑（仅工作日）

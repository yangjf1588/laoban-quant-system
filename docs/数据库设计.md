## 二、表结构设计

### 1. 股票列表表 (stock_list)

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | TEXT PRIMARY KEY | 股票代码 (e.g. 600000.SH) |
| symbol | TEXT | 股票代码 (e.g. 600000) |
| name | TEXT | 股票名称 |
| area | TEXT | 地域 |
| industry | TEXT | 所属行业 |
| market | TEXT | 市场类型 (主板/创业板/科创板) |
| list_date | TEXT | 上市日期 |
| created_at | TEXT | 创建时间 |

### 2. 日K线数据表 (daily_kline)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PRIMARY KEY | 自增ID |
| ts_code | TEXT | 股票代码 |
| trade_date | TEXT | 交易日期 |
| open | REAL | 开盘价 |
| high | REAL | 最高价 |
| low | REAL | 最低价 |
| close | REAL | 收盘价 |
| volume | REAL | 成交量 |
| amount | REAL | 成交额 |
| amplitude | REAL | 振幅 |
| pct_chg | REAL | 涨跌幅 |
| change | REAL | 涨跌额 |
| created_at | TEXT | 创建时间 |

**索引**: UNIQUE(ts_code, trade_date)

### 3. 实时行情表 (realtime_quote)

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | TEXT PRIMARY KEY | 股票代码 |
| name | TEXT | 股票名称 |
| price | REAL | 最新价 |
| volume | REAL | 成交量 |
| amount | REAL | 成交额 |
| bid1_vol | REAL | 买一量 |
| ask1_vol | REAL | 卖一量 |
| open | REAL | 开盘价 |
| high | REAL | 最高价 |
| low | REAL | 最低价 |
| close | REAL | 昨收价 |
| pct_chg | REAL | 涨跌幅 |
| update_time | TEXT | 更新时间 |
| created_at | TEXT | 创建时间 |

### 4. 涨停板记录表 (limit_up_record)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PRIMARY KEY | 自增ID |
| ts_code | TEXT | 股票代码 |
| name | TEXT | 股票名称 |
| trade_date | TEXT | 交易日期 |
| close_price | REAL | 涨停价 |
| pct_chg | REAL | 涨幅 |
| volume | REAL | 成交量 |
| amount | REAL | 成交额 |
| created_at | TEXT | 创建时间 |

**索引**: INDEX(ts_code, trade_date)

## 四、使用示例

```python
import sqlite3

conn = sqlite3.connect('data/database/quant.db')
cursor = conn.cursor()

# 查询某只股票最近30天K线
cursor.execute("""
    SELECT trade_date, open, high, low, close, volume
    FROM daily_kline
    WHERE ts_code = ?
    ORDER BY trade_date DESC
    LIMIT 30
""", ('600000.SH',))

# 查询近30天有涨停的股票
cursor.execute("""
    SELECT DISTINCT ts_code, name
    FROM limit_up_record
    WHERE trade_date >= date('now', '-30 days')
""")
```

## 一、T+1交易制度

### 核心规则
- **当天买入的股票，当天不能卖出**
- 资金当天可取出，但股票要第二个交易日才能卖
- **实行交割**：股票交割在成交后下一个交易日完成

### 对量化系统的影响
```python
# ❌ 错误：当天买的当天想卖
buy_stock("600000")
sell_stock("600000")  # 不允许！

# ✅ 正确：第二天才能卖
buy_stock("600000")
# 下一个交易日
sell_stock("600000")
```

### 系统必须做的
1. **持仓追踪**：记录每只股票的买入日期
2. **卖出限制检查**：只有上一交易日及之前买入的才能卖
3. **资金冻结**：买入时冻结资金，不是实时扣款

## 二、涨跌停板制度

### 主板（沪深主板）
- 普通股票：±10%
- ST股票：±5%
- 首个交易日无涨跌停限制

### 科创板、创业板
- 新股上市前5个交易日：无涨跌停
- 5个交易日后：±20%

### 北交所
- 新股上市首日：无涨跌停
- 之后：±30%

### 涨跌停封板逻辑
```
涨停 = 当日最高价 = 涨停价
跌停 = 当日最低价 = 跌停价

封板时间点：
- 早盘：9:30-10:00最常见（强势股）
- 午盘：11:00-13:30
- 尾盘：14:30-15:00（偷袭板）
```

### 对量化系统的警示
1. **涨停股买不进去**：按市价下单会被拒
2. **跌停股卖不出去**：流动性枯竭
3. **打板策略**需要判断封板时间

## 三、交易时间

### 正常交易时间
| 时段 | 时间 | 说明 |
|------|------|------|
| 集合竞价 | 9:15-9:25 | 开盘前定价，可撤单 |
| 连续竞价 | 9:30-11:30 | 上午交易 |
| 午间休市 | 11:30-13:00 | |
| 连续竞价 | 13:00-15:00 | 下午交易 |
| 收盘竞价 | 14:57-15:00 | 科创板/创业板 |

### 集合竞价规则
- **9:15-9:20**：可挂单可撤单
- **9:20-9:25**：可挂单不可撤单
- **9:25-9:30**：价格固定，不成交

### 对老板的意义
> 老板53%的交易在9:30-9:40完成
> 这是**集合竞价后连续竞价初期**，波动最大

## 四、交易费用

### 费用构成
| 费用 | 收取方式 | 典型费率 |
|------|----------|----------|
| 印花税 | 卖出时收取 | 0.1%（国家） |
| 过户费 | 买卖双向 | 0.002%（沪市） |
| 经手费 | 买卖双向 | 0.00696% |
| 证管费 | 买卖双向 | 0.002% |
| 佣金 | 买卖双向 | ≤0.03%，最低5元 |

### 实际计算示例
```
买入100万股票：
印花税：0（买入不收）
过户费：1000000 × 0.002% = 20元
经手费：1000000 × 0.00696% = 69.6元
证管费：1000000 × 0.002% = 20元
佣金：1000000 × 0.03% = 300元（最低5元）

买入总费用 ≈ 410元 ≈ 0.041%

卖出100万股票：
印花税：1000000 × 0.1% = 1000元
其他费用 ≈ 410元
卖出总费用 ≈ 1410元 ≈ 0.141%
```

### 系统必须做的
1. **精确计算交易成本**
2. **滑点预估**：考虑买卖价差
3. **印花税单独处理**：卖出时才收

## 五、融资融券基本规则

### 融资（做多）
- 借钱买股票
- 担保证券需折算

### 融券（做空）
- **A股普通账户不能裸卖空**
- 必须是融券标的（约1000多只大盘蓝筹）
- 归还时按市价结算

### 对老板策略的影响
> 老板说"做空受限"  
> 所以老板策略只能做多，不能做空

## 六、涨跌停相关量化指标

### 涨停家数（市场情绪指标）
```python
# 伪代码：计算涨停家数
limit_up_count = len(stocks_with_price_change == 10)  # 主板
limit_up_count_20 = len(stocks_with_price_change == 20)  # 创业板/科创板
```

### 炸板率（情绪转折）
```
炸板率 = 今日炸板股数 / 昨日涨停股数 × 100%

炸板率高 = 情绪开始退潮
```

### 一字板识别
```
开盘价 == 最高价 == 最低价 == 收盘价
且 涨停
= 一字板（散户买不进，次日可能开板）
```

## 七、老板交易特征对应的规则理解

| 老板特征 | 对应规则 | 系统注意事项 |
|----------|----------|--------------|
| 早盘10分钟决胜 | 9:30-9:35波动最大 | 开盘初期流动性最好 |
| 53%交易在此时 | 集合竞价后的窗口 | 快速下单能力 |
| 隔日超短 | T+1限制 | 持仓不能当天卖 |
| 不做尾盘 | 14:30后机会少 | 避免尾盘操作 |
| 近月有涨停 | 涨停板战法 | 识别涨停基因 |
| 单票底仓1成 | 仓位管理 | 单票不超过10万 |

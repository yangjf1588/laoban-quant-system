## 一、数据质量判断

### 好数据的标准
1. **完整性**：无缺失值
2. **准确性**：价格/成交量合理
3. **一致性**：格式统一
4. **时效性**：及时更新

### 常见数据问题

#### 1. 停牌数据
```python
# 停牌时价格不变，但成交量为0
df[df['volume'] == 0]  # 停牌股

# 处理方法：前值填充
df['close'] = df['close'].fillna(method='ffill')
```

#### 2. 涨跌停数据异常
```python
# 涨停时price变化应该是10%
# 如果>11%可能是数据错误

# 过滤异常涨跌幅
df = df[df['pct_chg'].abs() < 20]  # A股最大20%
```

#### 3. 除权除息数据
```python
# 复权处理
# 前复权(qfq)：以最新价格为基准调整历史
# 后复权(hfq)：以发行价为基准调整

# akshare获取复权数据
df = ak.stock_zh_a_hist(symbol="600000", adjust="qfq")  # 前复权
```

## 三、时序数据处理

### K线合成
```python
# 1分钟K线合成5分钟K线
def resample_kline(df_1min, period='5T'):
    df = df_1min.set_index('time')
    ohlc = {
        'open': 'first',
        'high': 'max',
        'low': 'min',
        'close': 'last',
        'volume': 'sum'
    }
    return df.resample(period).agg(ohlc)
```

### 移动平均
```python
# 计算MA5, MA10, MA20
df['MA5'] = df['close'].rolling(window=5).mean()
df['MA10'] = df['close'].rolling(window=10).mean()
df['MA20'] = df['close'].rolling(window=20).mean()
```

## 五、数据验证

### 基础验证
```python
def validate_data(df):
    errors = []
    
    # 1. 价格不能为0
    if (df['close'] == 0).any():
        errors.append("存在价格为0的记录")
    
    # 2. 涨跌幅应该在范围内
    if (df['pct_chg'].abs() > 20).any():
        errors.append("存在涨跌超20%的异常数据")
    
    # 3. 成交量不能为负
    if (df['volume'] < 0).any():
        errors.append("存在负成交量")
    
    # 4. 高价不能低于低价
    if (df['high'] < df['low']).any():
        errors.append("存在高价<低价的异常")
    
    return errors
```

### 报警机制
```python
def check_data_health():
    """每小时检查一次"""
    errors = validate_data(get_today_data())
    if errors:
        send_alert(f"数据异常: {errors}")
```

## 六、性能优化

### 批量写入
```python
# ❌ 慢：逐条写入
for row in df.iterrows():
    insert_db(row)

# ✅ 快：批量写入
df.to_sql('table', conn, if_exists='append', method='multi')
```

### 增量更新
```python
# 只更新今天的数据
today = datetime.now().strftime('%Y%m%d')
df_today = df[df['date'] == today]
df_today.to_sql('table', conn, if_exists='replace')
```

### 索引优化
```sql
-- 创建索引加速查询
CREATE INDEX IF NOT EXISTS idx_trade_date ON daily_kline(trade_date);
```

# 乌龙指防范模块 使用说明

**版本**: V1.0
**时间**: 2026-04-23
**作者**: 大龙

---

## 功能特性

| 功能 | 说明 | 状态 |
|------|------|------|
| 代码相似度 | 输入代码，找出形近代码，防止买错 | ✅ |
| ST股票识别 | 标记*ST股票，禁止买入 | ✅ |
| 金额超限 | 超设定限额弹窗确认 | ✅ |

---

## 快速开始

```python
from src.risk.fat_finger_guard import FatFingerGuard

guard = FatFingerGuard()

# 完整检查
result = guard.full_check("600552", price=10.0, quantity=10000)
print(result)

# 单项检查
guard.is_st_stock("600552")      # ST股票？
guard.check_code_similarity("600552")  # 相似代码
guard.check_amount_limit("600000", 10.0, 100000)  # 金额超限？
```

## 使用场景

### 1. 下单前检查
```python
# 老板下单前
result = guard.full_check(code, price, quantity)
if not result['pass']:
    print("⚠️ 有警告，确认是否继续？")
    for w in result['warnings']:
        print(w['message'])
```

### 2. 选股过滤
```python
# 过滤ST股票
for code in candidate_list:
    is_st, name = guard.is_st_stock(code)
    if is_st:
        print(f"排除ST: {name}")
```
